多维度软件视图

按阶段划分： build and run-time vies (构造时/运行时视图)
按动态性划分： moment and eriod views （时刻/阶段视图）
按构造对象划分： code and component views （代码/构件视图）

需要关注的是每个视图的元素、关系和模型

build-time

代码级视图：源代码 源代码如何通过基本程序块（如函数，类，方法，接口等）进行逻辑组织，以及它们之间依赖关系代码的逻辑组织
组件级视图：体系结构–源代码如何按文件、目录、包、库以及它们之间的一拉关系进行物理组织代码的物理组织
时刻视图：源代码和组件在特定时间内的样子特定时刻的软件形态
期间视图：它们如何随着时间的推移而发展/变化。软件形态随时间的变化

ArrayList 与 LinkedLIst

ArrayList 它长于随机访问元素，但是在List中间插入和删除元素比较慢（感觉看上去像是vector）
LinkedList 它通过代价较低的在List中间进行的插入和删除操作，提供了我优化的顺序访问，LinkedList在随机访问方面相对比较慢。

###ListIterator
ListIterator是一个更强大的Iterator的子类型，它只能用于各种List类的访问。
其特点是可以实现双向移动，同时可以使用set()的方法替换它访问过的最后一个元素

###Set
Set可以用contains函数判定一个set是否是另一个set的子集
